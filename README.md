# MovieSystem

MovieSystem is a full stack web application built with **ASP.NET Core (.NET 9)** and **Angular 19**.
The application allows users to search for cinemas, view movie listings and rate movies.

The project was developed with a real-world approach, focusing on clean architecture,
separation of concerns and modern web development practices.

- Search for cinemas and view available movie listings
- Movie rating system
- User authentication and authorization using JWT
- Role-based access control
- RESTful Web API built with ASP.NET Core
- Single Page Application (SPA) built with Angular
- Reactive forms and HTTP communication between frontend and backend
- UI components using Angular Material
- Map integration using Leaflet
- Spatial data storage using NetTopologySuite
- Image upload support (Azure Storage or local storage)
- Automated tests to ensure correct application behavior
